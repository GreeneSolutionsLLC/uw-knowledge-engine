# Contributing to Unfiltered Wisdom Machine-Readable Version

Thank you for your interest in improving this resource for trauma-aware AI development.

## How to Contribute

### Reporting Issues

If you find:
* Inaccuracies in the frameworks
* Missing safety considerations
* Unclear explanations
* Technical errors

Please open an issue with:
* Clear description of the problem
* Location (chapter/section)
* Suggested improvement (if applicable)

### Suggesting New Prompt Frameworks

To suggest a new human prompt framework:

1. **Check existing prompts** - Make sure it's not already covered
2. **Follow the template** - Use the structure from existing prompt files
3. **Include all sections**:
   - Relevant concepts
   - AI response framework
   - Validation approach
   - Safety/escalation criteria
   - Related chapters
   - Attribution

4. **Test your framework** - Ensure it follows trauma-informed principles

### Improving Existing Content

When suggesting improvements:
* Maintain the non-pathologizing tone
* Keep explanations accessible (avoid jargon)
* Ensure safety guidelines are clear
* Preserve attribution to Rob Greene
* Follow CC BY 4.0 license requirements

## Content Guidelines

### Must Follow:
* **Trauma-informed** - Validate, don't pathologize
* **Non-diagnostic** - Never diagnose or label
* **Safety-first** - Clear escalation criteria
* **Accessible language** - Plain English, no jargon
* **Evidence-based** - Grounded in trauma research
* **Ethical boundaries** - Know AI limitations

### Must Avoid:
* Clinical diagnosis language
* "Fix yourself" messaging
* Toxic positivity
* Minimizing experiences
* Promising healing outcomes
* Replacing professional care

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test for consistency with existing content
5. Submit PR with clear description
6. Reference any related issues

## Code of Conduct

### Our Standards

* **Respectful** - Treat all contributors with respect
* **Inclusive** - Welcome diverse perspectives
* **Constructive** - Provide helpful feedback
* **Ethical** - Prioritize human safety and wellbeing
* **Collaborative** - Work together to improve the resource

### Unacceptable Behavior

* Harassment or discrimination
* Dismissing trauma experiences
* Promoting harmful practices
* Violating ethical guidelines
* Disrespecting the author's work

## Attribution Requirements

All contributions must:
* Maintain attribution to Rob Greene
* Preserve CC BY 4.0 license
* Link to original work: https://unfiltered-wisdom.com/the-book/
* Not remove or alter existing attributions

## Questions?

For questions about:
* **Content accuracy** - Reference the original book
* **Technical issues** - Open a GitHub issue
* **Licensing** - See LICENSE.md
* **Author contact** - Visit https://unfiltered-wisdom.com
